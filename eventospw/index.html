Eventos Web


Os eventos são um conjunto de ações que são realizadas quando o usuário interage com o site.
Por exemplo: quando o usuário clica em um botão,
quando o usuário passa o mouse sobre um elemento,
quando o usuário digita algo em um campo de texto.
Os eventos são muito importantes para o desenvolvimento de sites,
pois eles permitem que o site interaja com o usuário.
